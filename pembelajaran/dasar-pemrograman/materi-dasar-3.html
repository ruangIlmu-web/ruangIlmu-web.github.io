Apa Itu Coding dan Bagaimana Cara Mempelajarinya?

Dengan kemajuan teknologi, profesi programmer semakin banyak diminati oleh perusahaan. Bagi pemula yang tertarik untuk berkarir sebagai programmer, penting untuk mempelajari coding terlebih dahulu.

Apa Itu Coding?

Coding adalah proses menulis baris-baris kode dalam bahasa pemrograman. Tujuannya adalah untuk berkomunikasi dengan komputer agar dapat mengeksekusi perintah yang kita berikan. Berbeda dengan komunikasi manusia, komputer tidak memahami bahasa atau kata-kata; mereka hanya mengenal sinyal on dan off, yang diwakili oleh kode biner (1 dan 0). Kode biner ini dikelompokkan menjadi bytes, yaitu kelompok 8 digit yang merepresentasikan data.

Dengan banyaknya data yang harus diolah, manusia tidak bisa melakukannya secara manual karena akan memakan waktu dan rentan terhadap kesalahan. Oleh karena itu, bahasa pemrograman dibutuhkan untuk menyederhanakan pengelolaan data tersebut.

Proses Coding

Ketika kita menulis kode, kita menggunakan sintaks yang sesuai dengan bahasa pemrograman. Sintaks ini akan melalui proses yang disebut kompilasi, yang menjadikan kode lebih mudah dipahami oleh komputer. Ada beberapa jenis bahasa pemrograman berdasarkan proses kompilasinya:

  • Compiled Language:Bahasa yang harus diterjemahkan oleh compiler terlebih dahulu, seperti C, C++, atau Go.
  • Interpreted Language:Bahasa yang bisa langsung dieksekusi tanpa kompilasi, seperti JavaScript, PHP, atau Python.
  • Just-In-Time (JIT):Bahasa yang diterjemahkan oleh compiler saat program dijalankan, seperti Java dan C#.

Sintaks dan Semantik

Setiap bahasa pemrograman memiliki aturan penulisan yang berbeda. Sintaks adalah aturan penulisan karakter yang benar, sedangkan semantik adalah makna dari kode yang kita tulis. Kode bisa saja benar secara sintaks tetapi salah secara semantik.

  • Dalam bahasa C, untuk mencetak "Hello, World" kita menulis: 
                            #include 
                            int main() {
                                printf("Hello, World");
                                return 0;
                            }
  • Dalam bahasa Java: 
                            class Main {
                            public static void main(String[] args) {
                                System.out.println("Hello, World");
                            }
                        }
  • Dalam PHP: 
                            echo "Hello, World";

Statement dan Expression

Dalam coding, kita juga harus memahami perbedaan antara statement dan expression.

  • Statement: Unit sintaks yang menyatakan aksi, seperti mendeklarasikan variable.
  • Expression: Entitas yang menghasilkan nilai, seperti operasi matematika.

    • Contoh statement Javascript

      var x = 1; // Ini adalah statement

    • Contoh expression Javascript

      x + 1; // Ini adalah expression yang menghasilkan nilai baru

Penting untuk memahami bahwa sebuah statement dapat berisi expression. Misalnya, var x = 4 + 2; adalah statement yang juga mengandung expression 4 + 2.

Dengan memahami konsep dasar seperti sintaks, semantik, statement, dan expression, kalian bisa mulai belajar coding dengan lebih baik.
